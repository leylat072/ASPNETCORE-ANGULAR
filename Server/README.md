# Server


## Development server

Run 'dotnet restore'

Set 'MediKeeperContext' to your sql server database

Run `dotnet run` for a dev server. Navigate to `http://localhost:44390/`. 

## http://localhost:44390/item/items-> list of items

## http://localhost:44390/item/itembyid-> return item by id 

## http://localhost:44390/item/itembyname-> return item by name

## http://localhost:44390/item/delete/id-> delete item by id

## http://localhost:44390/item/itembyid-> return item by id 

## http://localhost:44390/item/maxitems-> return max items for each item

## http://localhost:44390/item/maxitems/name-> return max item in given name
